## はじめに

このリポジトリは、Takosukeが制作したMMDモデルが保管されています。  
以下の注意事項をよく読んでご利用下さい。  

**また、現在は自作エクスポーターを使用してPMXファイルを出力しています**  
**データに不具合がある可能性がありますので、モデルの使用についてはご注意ください。**
